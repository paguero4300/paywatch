# Sonidos de Notificación

## Instrucciones

Coloca un archivo de audio llamado `notification.mp3` en esta carpeta para reproducir un sonido cuando llegue un nuevo pago.

### Formato recomendado:
- **Archivo**: `notification.mp3`
- **Formato**: MP3
- **Duración**: 1-3 segundos
- **Tamaño**: Menos de 100KB

### Dónde conseguir sonidos gratuitos:
- https://pixabay.com/sound-effects/
- https://freesound.org/
- https://mixkit.co/free-sound-effects/

### Ejemplo de búsqueda:
- "notification sound"
- "ding sound"
- "cash register"
- "success sound"

Una vez descargado, renombra el archivo a `notification.mp3` y colócalo en esta carpeta.
